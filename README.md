# Практическая работа №5 — Starter Kit

Добро пожаловать!  
Это минимальный набор файлов для выполнения практической работы по теме  
**"Объектная модель браузера (BOM) и Window. Работа с таймерами и диалоговыми окнами"**.

---

## Что в комплекте

- **index.html** — готовая разметка с интерфейсом для трёх заданий.
- **style.css** — базовые стили для удобного и аккуратного отображения.
- **script.js** — минимальная заготовка с пустыми функциями и подсказками, где писать код.

---

## Задания

### 1. Секундомер

- Реализуйте секундомер с кнопками "Старт", "Стоп", "Сброс".
- Используйте `setInterval` для обновления времени каждую секунду.
- Формат времени: ЧЧ:ММ:СС (например, 00:01:25).
- Сохраняйте ID таймера в переменную, чтобы можно было остановить.
- При "Стоп" останавливайте таймер, при "Сброс" — обнуляйте счётчик.

### 2. Обратный отсчёт

- Пользователь вводит количество секунд (целое число от 1 до 3600).
- При нажатии "Старт" запускается обратный отсчёт.
- Отображайте оставшееся время в формате ЧЧ:ММ:СС.
- При достижении 00:00 показывайте alert "Время вышло!" и останавливайте таймер.
- Кнопка "Стоп" останавливает таймер досрочно.

### 3. Диалоговые окна

- Реализуйте кнопки для вызова `alert`, `confirm` и `prompt`.
- Для `confirm` и `prompt` выводите результат в блок под кнопками.
- Обрабатывайте отмену в `prompt` (если пользователь нажал "Отмена").

---

## Советы по выполнению

- Используйте `document.getElementById` для получения элементов.
- Для форматирования времени используйте `String(...).padStart(2, '0')`.
- Сохраняйте ID таймера, возвращаемый `setInterval`, чтобы потом остановить его через `clearInterval`.
- Проверяйте ввод пользователя на корректность (целое число, диапазон).
- Комментируйте код, объясняя _почему_ вы делаете так, а не просто _что_ делаете.
- Тестируйте каждую функцию отдельно.
- Не бойтесь использовать `console.log` для отладки.

---

## Как запустить

1. Скачайте все файлы в одну папку.
2. Откройте файл `index.html` в браузере.
3. Откройте DevTools (F12) для отладки и просмотра ошибок.
4. Пишите код в `script.js` в отведённых местах.
